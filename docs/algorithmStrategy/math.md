## 充分必要条件

如果命题 p 能推出命题 q，且命题 q 也能推出命题 p，则称 p 是 q 的充分必要条件，且 q 也是 p 的充分必要条件。

p 能推出 q，p 是 q 的充分条件；q 是否能推出 p 决定了 p 和 q 的必要条件关系。

- q 不能推出 p，则 p 是 q 的充分不必要条件。
- q 能推出 p，则 p 是 q 的充分必要条件。



## 集合

集合用二进制表示，二进制第 i 位为 1 表示在集合中，为 0 表示不在集合中。例如集合{0,2,3} 可以用二进制 `1101` 表示。非负整数的的集合可以压缩成一个数字。可以使用位运算高效的做一些集合有关的运算。



**集合与集合**

- 对称差：集合 A 与集合 B 中所有不属于 A ∩ B 的元素的集合。
- 差：A - B，属于 A 但是不属于 B 的的所有元素组成的集合，`A - B == ∅`(空集)，说明 A 是 B 的子集。



| 术语       | 集合         | 位运算                    | 集合示例                  | 位运算示例                                  |
| ---------- | ------------ | ------------------------- | ------------------------- | ------------------------------------------- |
| 交集       | A ∩ B        | a & b                     | {0,2} ∩ {0,2,3} = {0,2}   | 0101 & 1101 = 0101                          |
| 并集       | A ∪ B        | a \| b                    | {0,2} ∩ {0,2,3} = {0,2,3} | 0101 \| 1101 = 1101                         |
| 对称差     | A △ B        | a ^ b                     | {0,2} △ {0,2,3} = {3}     | 0101 ^ 1101 = 1000                          |
| 差         | A - B        | a & ~b                    | {0,2,3} - {0,1} = {2,3}   | 1101 & 1100 =1100                           |

## GCD

求两个数的最大公约数（最大公因数）。

**辗转相除法**：通过反复用余数替换较大数，直到余数为 0，此时的**除数**就是最大公约数。


~~~java
public int gcd(int a, int b) {
    // a > 0, b > 0;
    while (b != 0) {
        int temp = b;
        b = a % b;
        a = temp;
    }
    return a;
}
~~~

**更相减损法**：通过反复用两个数的差替换较大数，直到两数相等，这个相等数就是最大公约数。

如果两个数都是偶数，可以先除以 2 进行化简，同时记录除以 2 的次数。

~~~java
public int gcd(int a, int b) {
    // a > 0, b > 0;
    int factor = 1;
    while ((a & 1) == 0 && (b & 1) == 0) {
        a >>= 1;
        b >>= 1;
        factor <<= 1;
    }
    while (a != b) {
        if (a > b) {
            a = a - b;
        } else {
            b = a - b;
        }
    }
    return a * factor;
}
~~~



### gcd 和 lcm

Lowest Common Multiple：最小公倍数。

Greatest Common Divisor：最大公约数

lcm(a, b) * gcd(a, b) = a * b;

~~~md
证明过程: a 和 b 的最大公约数是 g, m 和 n 互质
a = g * m, m 为质数
b = g * n, n 为质数
lcm(a, b) = lcm(gm, gn) = g * m * n, 因为 m 和 n 互质
a * b = g^2 * m * n = g * g * m * n = gcd(a,b) * lcm(a,b)

~~~







## 上取整下取整公式转换证明

$$
\text{a 为非负整数, b 为正整数:} \lceil \frac{a}{b} \rceil = \lfloor \frac{a + b - 1}{b} \rfloor \\
\text{分类讨论证明:} \\
\text{当 a mod b = 0: } 
\lceil \frac{a}{b} \rceil  = \lfloor \frac{a + b - 1}{b} \rfloor 
= \lfloor \frac{a}{b} + \frac{b-1}{b} \rfloor
= \frac{a}{b} + \lfloor \frac{b-1}{b} \rfloor
= \frac{a}{b} \text{ 等式恒成立} \\
\text{当 a mod b > 0: }
\lceil \frac{a}{b} \rceil  = \lfloor \frac{a + b - 1}{b} \rfloor 
= \lfloor \frac{a-1}{b} + \frac{b}{b} \rfloor
= \lfloor \frac{a-1}{b} \rfloor + 1 \text{ 等式恒成立}
$$

### 加法和乘法的取模

如何计算 `1234 * 5678` 结果的个位数？

乘法运算，只有最低位会影响个位数结果，只需要考虑 `4 * 8` 的结果。

求 `1234 + 5678` 结果的个数位？

同理，只需要考虑 `4 + 8` 的结果。

~~~java
(a * b) % 10 = ((a % 10) * (b % 10)) % 10;
(a + b) % 10 = ((a % 10) + (b % 10)) % 10;
~~~

当模数不为 10 时，证明上面的等式成立。

~~~java
(a * b) % m = ((a % m) * (b % m)) % m;
(a + b) % m = ((a % m) + (b % m)) % m;
~~~

`(a * b) % m = ((a % m) * (b % m)) % m`  证明：

根据带余除法，任意整数 a 都可以表示位 a = qm + r（m != 0）,其中整数 q 位 a 除以 m 的商，整数 r 位 a 除以 m 的余数，即 r = a % m。

设整数 a = q1m + r1, b= q2m + r2。

~~~java
(a * b) % m
    = ((q1m + r1) * (q2m + r2)) % m
    = (q1q2m^2 + q1r2m + r1q2m + r1r2) % m
    = (r1 * r2) % m
    = ((a % m) * (b % m)) % m
~~~

~~~java
(a + b) % m
    = ((q1m + r1) + (q2m + r2)) % m
    = (r1 + r2) % m
    = ((a % m) + (b % m)) % m
~~~

根据这两个恒等式，对某个乘法和加法运算结果取模，可以在运算的过程中就进行取模。

**如果涉及到幂运算，指数是不能随意取模的。如果指数在 64 位整数的范围内，可以用快速幂计算。如果指数超出 64 位整数的范围，需欧拉降幂。**



### 同余

两个整数 a 和 b，如果 `(a - b) % m = 0` （a != b，a-b 是 m 的倍数），则称 a 与 b 关于 m 同余。记为 `a ≡ b (mod m)`。

`a ≡ b (mod m)` 成立，则 `ka ≡ kb (mod m)` 也成立。

**同余式移项：**

`a + b ≡ c + d (mod m)`，则 `a - c ≡ d - b (mod m)` 成立。

**证明**：a + b - (c + d) 是 m 的倍数，那么 a - c - (d - b) 也是 m 的倍数，则有 `a - c ≡ d - b (mod m)`。

`a ≡ b (mod m)` 成立，则 `a + x ≡ b + x (mod m)` 成立，在同余式两边加上（减去）同一个整数，同余式仍成立。

**证明**：左边加上 0 = x - x，`a + x - x ≡ b (mod m)`，根据同余式移项得 `a + x ≡ b + x (mod m)`。



### 负数和减法的取模

根据同余的定义，有 `-17 ≡ 3 (mod 10)`。

可以 -17 + 10 + 10 = 3，或者 -17 % 10 + 10 = -7 + 10 = 3。

一般地，如果 x < 0，0 < y < m，如果 `x ≡ y (mod m)`，即 x % m + m = y。

**如果计算过程中，有减法，可以在最终的结果做处理 (x % mod + x) % m。**



### 除法的取模

如果要计算 24 / 8 % 5 的结果。能像加法或乘法那样，写成 (24 % 5) / (8 % 5) = 4 / 3 吗？显然是不能的，结果都不是整数。

**结论**：如果 p 是质数，a 是 b 的倍数，且 b 和 p 互质（b 不是 p 的倍数），那么有 `a / b % p = (a * b ^(p-2) ) % p`。
$$
\text{引理 1：当 p 是质数且 1 ≤ i ≤ p−1 时，有：}
\\
\binom{p}{i} ≡ \text{0 (mod p)}
\\
\text{} \binom{p}{i} = \frac{p!}{(p-i)!i!}
$$
**证明**：当 `1 ≤ i ≤ p−1` 时，$\frac{p!}{(p-i)!i!}$ 的分母是不包含 p（且没有除了 1 之外的因子） 的。由于分子包含 p，且该结果为整数，所以 $\binom{p}{i}$ 可以被 p 整除。

**注**：如果 p 不是质数，p = 4，i = 2，$\binom{p}{i}$ = 6，不是 p = 4 （p 被分母的质因子拆分了）的倍数。
$$
\text{引理 2：对于任意整数 x 和 y 和任意质数 p，有 }\\
(x +y) ^ p = x^p + y^p \text{(mod p)} \\
\text{证明：}\\
(x + y)^p = \sum_{k=0}^{p} \binom{p}{k} x^{p-k}y^k\\
= \binom{p}{0} x^py^0 + \binom{p}{1} x^{p-1}y^1 + \binom{p}{2} x^{p-2}y^2 + \text{ ... } + \binom{p}{p-1} x^{1}y^{p-1} + \binom{p}{p} x^0y^p \text{ （二项式定理）} \\
根据\binom{p}{i} ≡ \text{0 (mod p)} \\
除了第一项和最后一项以外，其余项都是 p 的倍数，于是 \\
(x +y) ^ p ≡ x^p + y^p \text{(mod p)} \\
$$



**定理（费马小定理）**：对于任意整数 b 和 任意质数 p，有 `b^p ≡ b (mod p)`。

**数学归纳法证明**
$$
\text{证明：当 b = 0 时，0^p = 0 (mod p) 成立。} \\
\text{假设 b = k 时}，原命题成立，即 b^p = b (mod \; p)。\\
\text{根据引理 2，有：} \\
(k+1)^p ≡ k^p + 1^p \text{(mod p)} \\
\text{根据归纳假设得：} \\
(k+1)^p ≡ k + 1 \text{(mod p)}\\
\text{即当 b = k + 1 时，原命题 \quad b^k ≡ b (mod \; p)成立。}\\
根据数学归纳法，原命题对于 b≥0 成立。\\
\text{当 b < 0 时，假设 b = -k（k > 0 }\\
\text{当 p 是非 2 质数，奇数，有：} (-k)^p = -k^p \\
根据非负数情况： \quad k^p ≡ k (mod \; p) \\
同余式两边同时乘以 \text{ -1 }得：-k^p ≡ -k (mod \; p) \\
即 \quad b^p ≡ b (mod \;p) \\
当 \; p = 2 \; 时，(-k)^2 = k^2，根据定理 k^2 ≡ k \; (mod \; 2)，也成立。
$$
 **数论证明**
$$
如果 b \; 是 \; p \; 的倍数，则 \; b^k ≡ b \; (mod \; p) ≡ 0,显然成立。\\
如果 b \; 不是 \; p \; 的倍数，考虑集合 \; \{1, 2, ..., p-1\}，把他们都乘以 \; b，得到 \\
\{1b, 2b, ..., (p-1)b\}，\text{因为 p 是质数且，p 不能整除 b 所以新集合模 p 的余数恰好是} \\ 
\{1, 2, ..., p-1\} 的一个排列，因此 \\
\text{1 * 2 ... (p-1) ≡ b (2b) ... (p-1) b (mod p)} \\
(p-1)! ≡ b^{p-1} (p-1)! (mod \; p)
因为 (p-1)! 不是 p 的倍数，可以约去，得到 \quad b^{p-1} ≡ 1 (mod \; p) \\
两边再乘以 \; b，得到 \; b^p ≡ b (mod \; p)
$$


**除法取模证明**
$$
b^p ≡ b (mod \; p) 成立，根据同余式移项得：\\
b^p - b ≡ 0 (mod \; p) \quad 变形得 \quad  b(b^{(p-1)} - 1) ≡ 0 (mod \; p) \\
\text{如果 b 不是 p 的倍数，那么 } b^{p-1} - 1 \text { 必须是 p 的倍数（是倍数肯定成立），}，即 \\
b^{(p-1)} - 1 ≡ 0 (mod \; p)，移项得 \\
b^{(p-1)} ≡ 1 (mod \; p) \\
两边同时乘以 \; \frac{a}{b} （a 是 b 的倍数） \; 得 \; ab^{p-2} ≡ \frac{a}{b} (mod \; p) \\
即 \; \frac{a}{b} \bmod p = ab^{p-2} \bmod p
$$
所以：当 a 是 b 的倍数时，且 b 不是 p 的倍数，则`a / b % p = (a * b ^(p-2) ) % p`。

b 如果是 p 的倍数，右边的结果为 0 ，左边的结果不一定为 0。



### 逆元

当 a 不是 b 的倍数时，如果要求 a / b 要模 p。这个时候就需要用到逆元。

普通的实数运算中，一个数 a 的逆元就是它的倒数 `1/a`。

最大的特点：一个数乘以它的倒数，结果等于 1。

在模运算中，没有分数，怎么求 2 的模逆元呢？找到一个整数 x，让它来充当 1/2 这个角色。

x 需要满足一个条件 `2 * x ≡ 1 (mod p)`。

读作：2 乘以 x 后，对 p 取模的结果为 1。

整数 x，就是 2 在模 p下的乘法逆元。

求 a 模 p 的逆元，p 是质数。根据费马小定理有 `a^p ≡ a (mod p)`

得 `a^(p-2) ≡ 1/a (mod p)`
