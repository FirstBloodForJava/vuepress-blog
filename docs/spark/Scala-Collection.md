# Scala-Collection

从集合类的结构来看，集合可分为 3 个类别：

- `Sequence`：元素带有顺序的集合，indexed（数组）；linear（链表）
- `Map`：包含 key/value 键值对的集合
- `Set`：无序且唯一的集合

## 常用集合

| ------------- | ------ | ---- | ----------------------------------------------------- |
| `List`        | ✓      |      | 线性链表                                              |
| `Vector`      | ✓      |      | indexed                                               |
