## 使用

### 快捷键

1. 生成html格式的代码：! + Enter；
2. 代码格式化：Shift + Alt + F；
3. 向上或向下移动一行：Alt + Up/Down；
4. 快速复制一行：Shift + Alt + Up/Down
5. 快速保存(不保存浏览器不刷新)：Ctrl + S；
6. 快速查找：Ctrl + F；
7. 快速替换：Ctrl + H；
