JavaеӨҡзәҝзЁӢзҗҶи®ә

JavaеҶ…еӯҳжЁЎеһӢ

JavaеҶ…еӯҳжЁЎеһӢзҡ„дё»иҰҒзӣ®зҡ„жҳҜе®ҡд№үзЁӢеәҸдёӯеҗ„з§ҚеҸҳйҮҸзҡ„и®ҝй—®и§„еҲҷпјҢеҚіе…іжіЁеңЁиҷҡжӢҹжңәжҠҠеҸҳйҮҸеҖјеӯҳеӮЁеҲ°еҶ…еӯҳе’Ңд»ҺеҶ…еӯҳдёӯеҸ–еҮәеҸҳйҮҸеҖјиҝҷж ·зҡ„еә•еұӮз»ҶиҠӮгҖӮиҝҷйҮҢеҸҳйҮҸжңүе®һдҫӢеҸҳйҮҸгҖҒйқҷжҖҒеҸҳйҮҸгҖҒдёҚеҢ…жӢ¬еұҖйғЁеҸҳйҮҸе’Ңж–№жі•еҸӮж•°(еҸҳйҮҸдёәеј•з”Ёж—¶йҷӨеӨ–)пјҢеұҖйғЁеҸҳйҮҸе’Ңж–№жі•еҸӮж•°ж—¶зәҝзЁӢз§Ғжңүзҡ„пјҢдёҚдјҡиў«е…ұдә«гҖӮ

дё»еҶ…еӯҳе’Ңе·ҘдҪңеҶ…еӯҳ

JavaеҶ…еӯҳжЁЎеһӢи§„е®ҡдәҶжүҖжңүзҡ„еҸҳйҮҸйғҪеӯҳеӮЁеңЁдё»еҶ…еӯҳдёӯгҖӮжҜҸжқЎзәҝзЁӢжңүиҮӘе·ұзҡ„е·ҘдҪңеҶ…еӯҳпјҢзәҝзЁӢзҡ„е·ҘдҪңеҶ…еӯҳдё»иҰҒжҳҜдҝқеӯҳдәҶиў«иҜҘзәҝзЁӢдҪҝз”Ёзҡ„еҸҳйҮҸзҡ„дё»еҶ…еӯҳеүҜжң¬(е№¶дёҚдјҡжҳҜиҝҷдёӘеҜ№иұЎзҡ„жүҖжңүеҶ…е®№)пјҢзәҝзЁӢеҜ№еҸҳйҮҸзҡ„жүҖжңүж“ҚдҪң(иҜ»еҸ–гҖҒиөӢеҖј)йғҪеҝ…йЎ»еңЁе·ҘдҪңеҶ…еӯҳдёӯиҝӣиЎҢпјҢд№ҹдёҚиғҪд№Ӣй—ҙиҜ»еҶҷдё»еҶ…еӯҳдёӯзҡ„ж•°жҚ®гҖӮ

еҶ…еӯҳй—ҙдәӨдә’ж“ҚдҪң

дёҖдёӘеҸҳйҮҸеҰӮдҪ•д»Һдё»еҶ…еӯҳжӢ·иҙқеҲ°е·ҘдҪңеҶ…еӯҳгҖҒеҰӮдҪ•д»Һе·ҘдҪңеҶ…еӯҳеҗҢжӯҘдё»еҶ…еӯҳиҝҷдёҖзұ»зҡ„е®һзҺ°з»ҶиҠӮпјҢJavaеҶ…еӯҳжЁЎеһӢе®ҡд№үзҡ„8з§Қж“ҚдҪңжқҘе®ҢжҲҗгҖӮJavaиҷҡжӢҹжңәе®һзҺ°ж—¶пјҢеҝ…йЎ»дҝқиҜҒlockгҖҒunlockгҖҒreadгҖҒloadгҖҒuseгҖҒassignгҖҒstoreгҖҒwriteиҝҷдәӣж“ҚдҪңзҡ„еҺҹеӯҗжҖ§гҖҒдёҚеҸҜеҶҚеҲҶ(doubleе’Ңlongзұ»еһӢйҷӨеӨ–)гҖӮ

  • lock(й”Ғе®ҡ)пјҡдҪңз”ЁдәҺдё»еҶ…еӯҳзҡ„еҸҳйҮҸпјҢе®ғжҠҠдёҖдёӘеҸҳйҮҸж ҮиҜҶдёәдёҖдёӘзәҝзЁӢзӢ¬еҚ зҡ„зҠ¶жҖҒгҖӮ
  • unlock(и§Јй”Ғ)пјҡдҪңз”ЁдәҺдё»еҶ…еӯҳзҡ„еҸҳйҮҸпјҢе®ғжҠҠдёҖдёӘеӨ„дәҺй”Ғе®ҡзҠ¶жҖҒзҡ„еҸҳйҮҸйҮҠж”ҫеҮәжқҘпјҢйҮҠж”ҫд№ӢеҗҺжүҚиғҪиў«е…¶е®ғзәҝзЁӢй”Ғе®ҡгҖӮ
  • read(иҜ»еҸ–)пјҡдҪңз”ЁдәҺдё»еҶ…еӯҳзҡ„еҸҳйҮҸпјҢе®ғжҠҠдёҖдёӘдё»еҶ…еӯҳзҡ„еҸҳйҮҸдј иҫ“еҲ°зәҝзЁӢзҡ„е·ҘдҪңеҶ…еӯҳдёӯпјҢд»ҘдҫҝйҡҸеҗҺзҡ„loadдҪҝз”ЁгҖӮ
  • load(иҪҪе…Ҙ)пјҡдҪңз”ЁдәҺе·ҘдҪңеҶ…еӯҳпјҢе®ғе°Ҷreadд»Һдё»еҶ…еӯҳеҫ—еҲ°зҡ„еҸҳйҮҸеҖјж”ҫе…Ҙе·ҘдҪңеҶ…еӯҳзҡ„еҸҳйҮҸеүҜжң¬дёӯгҖӮ
  • use(дҪҝз”Ё)пјҡдҪңз”ЁдәҺе·ҘдҪңеҶ…еӯҳпјҢе®ғжҠҠе·ҘдҪңеҶ…еӯҳдёӯдёҖдёӘеҸҳйҮҸзҡ„еҖјдј йҖ’з»ҷжү§иЎҢеј•ж“ҺпјҢдҪҝз”ЁеҸҳйҮҸеҖјзҡ„еӯ—иҠӮз ҒжҢҮд»ӨдјҡдҪҝз”ЁиҝҷдёӘж“ҚдҪңгҖӮ
  • assign(иөӢеҖј)пјҡдҪңз”ЁдәҺе·ҘдҪңеҶ…еӯҳпјҢе®ғжҠҠдёҖдёӘжү§иЎҢеј•ж“ҺжҺҘж”¶зҡ„еҖјиөӢз»ҷе·ҘдҪңеҶ…еӯҳзҡ„еҸҳйҮҸпјҢиөӢеҖјеҸҳйҮҸзҡ„еӯ—иҠӮз ҒжҢҮд»ӨдјҡдҪҝз”ЁиҝҷдёӘж“ҚдҪңгҖӮ
  • store(еӯҳеӮЁ)пјҡдҪңз”ЁдәҺе·ҘдҪңеҶ…еӯҳпјҢе®ғжҠҠе·ҘдҪңеҶ…еӯҳдёӯзҡ„дёҖдёӘеҸҳйҮҸзҡ„еҖјдј иҫ“еҲ°дё»еҶ…еӯҳдёӯпјҢд»ҘдҫҝйҡҸеҗҺзҡ„writeдҪҝз”ЁгҖӮ
  • write(еҶҷе…Ҙ)пјҡдҪңз”ЁдәҺдё»еҶ…еӯҳзҡ„еҸҳйҮҸпјҢе®ғжҠҠstoreж“ҚдҪңиҺ·еҸ–еҲ°зҡ„еҖјж”ҫеҲ°дё»еҶ…еӯҳдёӯеҺ»гҖӮ

дё»еҶ…еӯҳжӢ·иҙқиҮіе·ҘдҪңеҶ…еӯҳпјҡreadгҖҒloadпјӣ

е·ҘдҪңеҶ…еӯҳеҗҢжӯҘеӣһдё»еҶ…еӯҳпјҡstoreгҖҒwriteпјӣ

дёҠиҝ°дёӨдёӘж“ҚдҪңиҰҒдҝқиҜҒйЎәеәҸжү§иЎҢпјҢдҪҶжҳҜдёҚдҝқиҜҒиҝһз»ӯжү§иЎҢгҖӮ

8з§Қж“ҚдҪңйңҖиҰҒж»Ўи¶ідёҖдёӢи§„еҲҷпјҡ

  • дёҚе…Ғи®ёreadе’ҢloadгҖҒstoreе’Ңwriteж“ҚдҪңд№ӢдёҖеҚ•зӢ¬еҮәзҺ°пјӣ
  • дёҚе…Ғи®ёдёҖдёӘзәҝзЁӢдёўејғе®ғжңҖиҝ‘зҡ„assignж“ҚдҪңпјӣ
  • дёҚе…Ғи®ёдёҖдёӘзәҝзЁӢж— еҺҹеӣ ең°(жІЎжңүеҸ‘з”ҹassignж“ҚдҪң)жҠҠж•°жҚ®д»Һе·ҘдҪңеҶ…еӯҳеҗҢжӯҘеӣһдё»еҶ…еӯҳпјӣ
  • ж–°зҡ„еҸҳйҮҸеҸӘиғҪеңЁдё»еҶ…еӯҳдёӯиҜһз”ҹпјӣ
  • дёҖдёӘеҸҳйҮҸеңЁеҗҢдёҖж—¶еҲ»еҸӘиғҪиў«еҗҢдёҖзәҝзЁӢиҝӣиЎҢlockж“ҚдҪң(еӨҡж¬ЎlockеӨҡж¬Ўunlock)пјӣ
  • еҰӮжһңеҜ№еҸҳйҮҸиҝӣиЎҢlockж“ҚдҪңпјҢйӮЈд№Ҳдјҡжё…з©әе·ҘдҪңеҶ…еӯҳдёӯжӯӨеҸҳйҮҸзҡ„еҖјпјҢжү§иЎҢеј•ж“ҺеңЁдҪҝз”ЁиҝҷдёӘеҸҳйҮҸеүҚпјҢйңҖиҰҒйҮҚж–°жү§иЎҢloadжҲ–assignж“ҚдҪңд»ҘеҲқе§ӢеҢ–еҸҳйҮҸзҡ„еҖјпјӣ
  • еҸҳйҮҸжІЎжңүиў«lockпјҢе°ұдёҚиғҪжү§иЎҢunlockпјҢдёҚе…Ғи®ёunlockеҺ»жү§иЎҢе…¶е®ғзәҝзЁӢй”Ғе®ҡзҡ„еҸҳйҮҸпјӣ
  • еҜ№дёҖдёӘеҸҳйҮҸжү§иЎҢunlockж“ҚдҪңд№ӢеүҚпјҢйңҖиҰҒжҠҠе·ҘдҪңеҶ…еӯҳзҡ„еҸҳйҮҸеҗҢдёҚеӣһдё»еҶ…еӯҳдёӯгҖӮ

з®ҖеҢ–ж“ҚдҪңдёәпјҡreadгҖҒwriteгҖҒlockе’Ңunlockеӣӣз§ҚгҖӮ

volatileзү№ж®Ҡи§„еҲҷ

volatileжҳҜJavaиҷҡжӢҹжңәжҸҗдҫӣжңҖиҪ»йҮҸзә§зҡ„еҗҢжӯҘжңәеҲ¶гҖӮ

volatileдёӨеӨ§зү№жҖ§пјҡ

  • еҸҜи§ҒжҖ§пјҡдҝқиҜҒеҸҳйҮҸеҜ№жүҖжңүзәҝзЁӢзҡ„еҸҜи§ҒжҖ§(еҪ“дёҖдёӘзәҝзЁӢдҝ®ж”№дәҶеҸҳйҮҸзҡ„еҖјпјҢж–°еҖјеҜ№е…¶е®ғзәҝзЁӢз«ӢеҚіеҸҜи§Ғ)
  • зҰҒжӯўжҢҮд»ӨйҮҚжҺ’пјҡдҝқиҜҒдёҖдёӘзәҝзЁӢйҮҢд»Јз Ғзҡ„дёІиЎҢжү§иЎҢпјҢжҢүд»Јз Ғзҡ„д№ҰеҶҷйЎәеәҸжү§иЎҢгҖӮ

volatileи§„еҲҷиҰҒжұӮпјҡ

жҜҸж¬ЎдҪҝз”ЁvolatileеҸҳйҮҸйғҪйңҖиҰҒд»Һдё»еҶ…еӯҳеҲ·ж–°жңҖж–°еҖјеҲ°е·ҘдҪңеҶ…еӯҳпјӣ

жҜҸж¬ЎvolatileеҸҳйҮҸзҡ„дҝ®ж”№йғҪдјҡиў«еҗҢжӯҘеҲ°дё»еҶ…еӯҳдёӯгҖӮ

иҝҷж ·volatileдҝ®йҘ°еҸҳйҮҸзҡ„i++ж“ҚдҪңе°ұдёҚжҳҜзәҝзЁӢе®үе…Ёзҡ„гҖӮ

public class VolatileTest {
    private static int i = 0;

    public static void main(String[] args) {

        Thread t1 = new Thread(() -> {
            try {
                Thread.sleep(1000);
            } catch (InterruptedException e) {
                throw new RuntimeException(e);
            }
            System.out.println(Thread.currentThread().getName() + "дҝ®ж”№жҲҗеҠҹ");
            add();
        }, "T1");

        Thread t2 = new Thread(() -> {
            while (i == 0) {
                // дёҚеҠ д»»дҪ•д»Јз ҒиҝҷйҮҢдјҡжӯ»еҫӘзҺҜ
            }
            System.out.println(Thread.currentThread().getName() +  " stop");
        }, "T2");

        t1.start();
        t2.start();

    }

    public static void add(){
        i++;
    }
}

еҺҹеӯҗжҖ§гҖҒеҸҜи§ҒжҖ§е’ҢжңүеәҸжҖ§

еҺҹеӯҗжҖ§

еҺҹеӯҗжҖ§еҸҳйҮҸж“ҚдҪңпјҡreadгҖҒloadгҖҒuseгҖҒassignгҖҒstoreе’ҢwriteгҖӮ

еҹәжң¬ж•°жҚ®зұ»еһӢзҡ„и®ҝй—®пјҢиҜ»еҶҷйғҪжҳҜе…·еӨҮеҺҹеӯҗжҖ§зҡ„(longе’Ңdoubleзҡ„йқһеҺҹеӯҗеҚҸе®ҡ64дҪҚ/32*2)гҖӮ

йңҖиҰҒжҸҗдҫӣжӣҙеӨ§и®ҝй—®зҡ„еҺҹеӯҗжҖ§ж“ҚдҪңпјҢJavaжІЎжңүзӣҙжҺҘжҸҗдҫӣlockе’Ңunlockж“ҚдҪңпјҢдҪҶжҳҜжҸҗдҫӣдәҶеӯ—иҠӮз ҒжҢҮд»Өmonitorenterе’ҢmonitorexitжқҘйҡҗејҸзҡ„дҪҝз”ЁиҝҷдёӨдёӘж“ҚдҪңгҖӮиҝҷдёӨдёӘж“ҚдҪңеҜ№еә”зҡ„е°ұжҳҜsynchronizedгҖӮ

еҸҜи§ҒжҖ§

еҪ“дёҖдёӘзәҝзЁӢдҝ®ж”№дёҖдёӘе…ұдә«еҸҳйҮҸд№ӢеҗҺпјҢе…¶е®ғзәҝзЁӢиғҪз«ӢеҚіеҫ—зҹҘиҝҷдёӘдҝ®ж”№гҖӮ

йҷӨдәҶvolatileиғҪдҝқиҜҒеҸҜи§ҒжҖ§пјҢJavaзҡ„synchronizedе’Ңfinalд№ҹиғҪе®һзҺ°еҸҜи§ҒжҖ§гҖӮ

synchronizedзҡ„еҸҜи§ҒжҖ§жҳҜйҖҡиҝҮеҜ№дёҖдёӘеҸҳйҮҸжү§иЎҢunlockж“ҚдҪңд№ӢеүҚпјҢйңҖиҰҒе°ҶиҝҷдёӘеҸҳйҮҸеҗҢжӯҘеӣһдё»еҶ…еӯҳдёӯгҖӮ

finalзҡ„еҸҜи§ҒжҖ§жҳҜпјҡиў«finalдҝ®йҘ°зҡ„еҸҳйҮҸ(йқҷжҖҒеҸҳйҮҸгҖҒе®һдҫӢеҸҳйҮҸ)пјҢеҸҳйҮҸдёҖдҪҶеҲқе§ӢеҢ–е®ҢжҲҗ(жІЎжңүthisеј•з”ЁйҖғйҖё)пјҢе°ұиғҪиў«е…¶е®ғзәҝзЁӢзңӢи§Ғfinalеӯ—ж®өзҡ„еҖјгҖӮ

еҜ№иұЎеҲӣе»әзҡ„иҝҮзЁӢпјҡ1гҖҒеҶҚе ҶдёӯеҲҶй…ҚеҶ…еӯҳпјӣ2гҖҒи°ғз”Ёжһ„йҖ еҷЁеҲӣе»әе®һдҫӢпјӣ3гҖҒе°Ҷеј•з”ЁжҢҮеҗ‘е®һдҫӢзҡ„еҶ…еӯҳгҖӮ

жңүеәҸжҖ§

volatileйҖҡиҝҮзҰҒжӯўжҢҮд»ӨйҮҚжҺ’дҝқиҜҒжңүеәҸжҖ§(еңЁеҚ•дёӘзәҝзЁӢеҶ…)пјӣ

synchronizedйҖҡиҝҮеҗҢдёҖж—¶еҲ»еҸӘиғҪжңүдёҖдёӘзәҝзЁӢеҜ№еҸҳйҮҸиҝӣиЎҢlockж“ҚдҪңпјҢжҺ§еҲ¶е…¶е®ғзәҝзЁӢеҸӘиғҪдёІиЎҢең°жү§иЎҢгҖӮ

е…ҲиЎҢеҸ‘з”ҹеҺҹеҲҷ

ж“ҚдҪңAе…ҲиЎҢеҸ‘з”ҹдәҺж“ҚдҪңBпјҢе…¶е®һе°ұжҳҜиҜҙеҸ‘з”ҹж“ҚдҪңBд№ӢеүҚпјҢж“ҚдҪңAдә§з”ҹзҡ„еҪұе“ҚиғҪиў«ж“ҚдҪңBи§ӮеҜҹеҲ°гҖӮ

еӨ©з„¶зҡ„е…ҲиЎҢеҸ‘з”ҹи§„еҲҷпјҡ

зЁӢеәҸж¬ЎеәҸи§„еҲҷпјҡеңЁдёҖдёӘзәҝзЁӢеҶ…пјҢжҢүз…§жҺ§еҲ¶жөҒзЁӢпјҢд№ҰеҶҷеңЁеүҚйқўзҡ„д»Јз Ғе…ҲиЎҢеҸ‘з”ҹдәҺд№ҰеҶҷеңЁеҗҺйқўзҡ„д»Јз ҒгҖӮ

зәҝзЁӢй”Ғе®ҡи§„еҲҷпјҡдёҖдёӘunlockж“ҚдҪңе…ҲиЎҢеҸ‘з”ҹдәҺеҗҺйқўеҜ№еҗҢдёҖдёӘй”Ғзҡ„lockж“ҚдҪңгҖӮеҚіеүҚдёҖдёӘзәҝзЁӢй”Ғе®ҡеҸҳйҮҸзҡ„дҝ®ж”№пјҢеҜ№дёӢдёҖдёӘиҺ·еҸ–й”Ғзҡ„зәҝзЁӢеҸҜи§ҒгҖӮ

volatileеҸҳйҮҸи§„еҲҷпјҡеҜ№дёҖдёӘvolatileеҸҳйҮҸзҡ„еҶҷж“ҚдҪңе…ҲиЎҢеҸ‘з”ҹдәҺеҗҺйқўеҜ№иҝҷдёӘеҸҳйҮҸзҡ„иҜ»ж“ҚдҪңгҖӮ

зәҝзЁӢеҗҜеҠЁи§„еҲҷпјҡThreadеҜ№иұЎзҡ„start()ж–№жі•е…ҲиЎҢеҸ‘з”ҹдәҺжӯӨзәҝзЁӢзҡ„д»»дҪ•ж“ҚдҪңгҖӮ

зәҝзЁӢз»Ҳжӯўи§„еҲҷпјҡзәҝзЁӢдёӯзҡ„жүҖжңүж“ҚдҪңйғҪе…ҲиЎҢеҸ‘з”ҹдәҺеҜ№жӯӨзәҝзЁӢзҡ„з»ҲжӯўжЈҖжөӢгҖӮ

зәҝзЁӢдёӯж–ӯи§„еҲҷпјҡзәҝзЁӢзҡ„interrupt()ж–№жі•и°ғз”Ёе…ҲиЎҢеҸ‘з”ҹдәҺиў«дёӯж–ӯзәҝзЁӢзҡ„д»Јз ҒжЈҖжөӢеҲ°дёӯж–ӯдәӢд»¶зҡ„еҸ‘з”ҹгҖӮ

еҜ№иұЎз»Ҳз»“еҺҹеҲҷпјҡдёҖдёӘеҜ№иұЎеҲқе§ӢеҢ–е®ҢжҲҗе…ҲиЎҢеҸ‘з”ҹдәҺе®ғзҡ„finalize()ж–№жі•зҡ„ејҖе§ӢгҖӮ

дј йҖ’жҖ§пјҡеҰӮжһңж“ҚдҪңAе…ҲиЎҢеҸ‘з”ҹдәҺж“ҚдҪңBпјҢж“ҚдҪңBе…ҲиЎҢеҸ‘з”ҹдәҺCпјҢйӮЈе°ұеҸҜд»Ҙеҫ—еҮәж“ҚдҪңAе…ҲиЎҢеҸ‘з”ҹдәҺBгҖӮ

зәҝзЁӢ

зәҝзЁӢжҳҜжҜ”иҝӣзЁӢжӣҙиҪ»йҮҸзә§зҡ„жү§иЎҢеҚ•дҪҚпјҢзәҝзЁӢзҡ„еј•е…ҘпјҢеҸҜд»Ҙе°ҶдёҖдёӘиҝӣзЁӢиө„жәҗзҡ„иө„жәҗеҲҶй…Қе’Ңжү§иЎҢи°ғеәҰеҲҶејҖпјҢеҗ„дёӘзәҝзЁӢеҸҜд»Ҙе…ұдә«иҝӣзЁӢиө„жәҗпјҢеҸҲеҸҜд»ҘзӢ¬з«Ӣи°ғеәҰгҖӮ

зәҝзЁӢзӣ®еүҚжҳҜJavaйҮҢйқўиҝӣиЎҢеӨ„зҗҶеҷЁиө„жәҗи°ғеәҰзҡ„жңҖеҹәжң¬еҚ•дҪҚгҖӮ

е®һзҺ°зәҝзЁӢзҡ„дёүз§Қж–№ејҸпјҡдҪҝз”ЁеҶ…ж ёзәҝзЁӢе®һзҺ°(1:1е®һзҺ°)гҖҒдҪҝз”Ёз”ЁжҲ·зәҝзЁӢе®һзҺ°(1:Nе®һзҺ°)гҖҒдҪҝз”Ёз”ЁжҲ·зәҝзЁӢдёҺиҪ»йҮҸзә§иҝӣзЁӢж··еҗҲе®һзҺ°(N:Mе®һзҺ°)гҖӮ

еҶ…ж ёзәҝзЁӢе®һзҺ°

еҶ…ж ёзәҝзЁӢ(KLT)е°ұжҳҜеҸҲж“ҚдҪңзі»з»ҹеҶ…ж ёж”ҜжҢҒзҡ„зәҝзЁӢпјҢиҝҷз§ҚзәҝзЁӢз”ұеҶ…ж ёжқҘе®ҢжҲҗзәҝзЁӢзҡ„еҲҮжҚўпјҢеҶ…ж ёйҖҡиҝҮж“ҚдҪңи°ғеәҰеҷЁеҜ№зәҝзЁӢиҝӣиЎҢи°ғеәҰпјҢе№¶иҙҹиҙЈе°ҶзәҝзЁӢзҡ„д»»еҠЎжҳ е°„еҲ°еҗ„дёӘеӨ„зҗҶеҷЁдёҠгҖӮ

зЁӢеәҸдёҖиҲ¬дёҚдјҡзӣҙжҺҘдҪҝз”ЁеҶ…ж ёзәҝзЁӢпјҢиҖҢжҳҜдҪҝз”ЁеҶ…ж ёзәҝзЁӢзҡ„дёҖз§Қй«ҳзә§жҺҘеҸЈ-иҪ»йҮҸзә§иҝӣзЁӢ(LWP)пјҢиҪ»йҮҸзә§иҝӣзЁӢе°ұжҳҜжҲ‘д»¬йҖҡеёёжүҖиҜҙзҡ„зәҝзЁӢпјҢз”ұдәҺжҜҸдёӘиҪ»йҮҸзә§иҝӣзЁӢйғҪжңүдёҖдёӘеҶ…ж ёзәҝзЁӢж”ҜжҢҒпјҢиҝҷз§ҚиҪ»йҮҸзә§иҝӣзЁӢе’ҢеҶ…ж ёзәҝзЁӢ1:1зҡ„е…ізі»з§°дёәдёҖеҜ№дёҖзҡ„зәҝзЁӢжЁЎеһӢгҖӮ

з”ұдәҺеҶ…ж ёзәҝзЁӢзҡ„ж”ҜжҢҒпјҢжҜҸдёӘиҪ»йҮҸзә§иҝӣзЁӢйғҪжҲҗдёәдәҶдёҖдёӘзӢ¬з«Ӣзҡ„и°ғеәҰеҚ•е…ғпјҢеҚідҪҝе…¶дёӯжҹҗдёҖдёӘиҪ»йҮҸзә§иҝӣзЁӢеңЁзі»з»ҹи°ғз”Ёдёӯиў«йҳ»еЎһдәҶпјҢд№ҹдёҚдјҡеҪұе“Қж•ҙдёӘиҝӣзЁӢзҡ„з»§з»ӯе·ҘдҪңгҖӮ

еұҖйҷҗжҖ§пјҡз”ұдәҺжҳҜеҹәдәҺеҶ…ж ёзәҝзЁӢе®һзҺ°зҡ„пјҢжүҖд»Ҙеҗ„з§ҚзәҝзЁӢж“ҚдҪңпјҢеҰӮеҲӣе»әгҖҒжһҗжһ„е’ҢеҗҢжӯҘпјҢйғҪйңҖиҰҒиҝӣиЎҢзі»з»ҹи°ғз”ЁгҖӮиҖҢзі»з»ҹи°ғз”Ёзҡ„д»Јд»·зӣёеҜ№иҫғй«ҳпјҢйңҖиҰҒеңЁз”ЁжҲ·жҖҒе’ҢеҶ…ж ёжҖҒдёӯжқҘеӣһеҲҮжҚўгҖӮиҪ»йҮҸзә§иҝӣзЁӢйңҖиҰҒеҶ…ж ёзәҝзЁӢж”ҜжҢҒпјҢж¶ҲиҖ—дёҖе®ҡзҡ„еҶ…ж ёиө„жәҗпјҢж”ҜжҢҒзҡ„иҪ»йҮҸзә§иҝӣзЁӢж•°йҮҸжңүйҷҗгҖӮ

з”ЁжҲ·зәҝзЁӢе®һзҺ°

е№ҝд№үдёҠпјҡдёҖдёӘзәҝзЁӢеҸӘиҰҒдёҚжҳҜеҶ…ж ёзәҝзЁӢпјҢйғҪеҸҜд»Ҙи®ӨдёәжҳҜз”ЁжҲ·зәҝзЁӢгҖӮ

зӢӯд№үдёҠпјҡз”ЁжҲ·зәҝзЁӢ(UT)жҳҜжҢҮе®Ңе…Ёе»әз«ӢеңЁз”ЁжҲ·з©әй—ҙзҡ„зәҝзЁӢеә“дёҠпјҢзі»з»ҹеҶ…ж ёдёҚиғҪж„ҹзҹҘеҲ°з”ЁжҲ·зәҝзЁӢзҡ„еӯҳеңЁеҸҠеҰӮдҪ•е®һзҺ°зҡ„гҖӮз”ЁжҲ·зәҝзЁӢзҡ„е»әз«ӢгҖҒеҗҢжӯҘгҖҒй”ҖжҜҒе’Ңи°ғеәҰйғҪеңЁз”ЁжҲ·жҖҒдёӯе®ҢжҲҗпјҢдёҚйңҖиҰҒеҶ…ж ёзҡ„её®еҠ©гҖӮ

еҰӮжһңзЁӢеәҸе®һзҺ°еҫ—еҪ“пјҢиҝҷз§ҚзәҝзЁӢдёҚйңҖиҰҒеҲҮжҚўеҲ°еҶ…ж ёжҖҒпјҢеӣ жӯӨж“ҚдҪңеҸҜд»ҘжҳҜйқһеёёеҝ«йҖҹдё”дҪҺж¶ҲиҖ—зҡ„пјҢд№ҹиғҪж”ҜжҢҒи§„жЁЎжӣҙеӨ§зҡ„зәҝзЁӢж•°йҮҸпјҢиҝҷз§ҚиҝӣзЁӢдёҺз”ЁжҲ·иҝӣзЁӢжҳҜ1:Nзҡ„е…ізі»пјҢжҲҗдёәдёҖеҜ№еӨҡзҡ„зәҝзЁӢжЁЎеһӢгҖӮ

ж··еҗҲе®һзҺ°

ж—ўеӯҳеңЁз”ЁжҲ·зәҝзЁӢпјҢд№ҹеӯҳеңЁиҪ»йҮҸзә§иҝӣзЁӢгҖӮз”ЁжҲ·зәҝзЁӢе»әз«ӢеңЁз”ЁжҲ·з©әй—ҙдёӯпјҢеӣ жӯӨз”ЁжҲ·зәҝзЁӢзҡ„еҲӣе»әгҖҒеҲҮжҚўгҖҒжһ„жһҗзӯүж“ҚдҪңдҫқз„¶е»үд»·пјҢе№¶дё”еҸҜд»ҘеӨ§и§„жЁЎзҡ„з”ЁжҲ·зәҝзЁӢе№¶еҸ‘пјҢиҖҢж“ҚдҪңзі»з»ҹж”ҜжҢҒиҪ»йҮҸзә§иҝӣзЁӢеҲҷдҪңдёәз”ЁжҲ·зәҝзЁӢе’ҢеҶ…ж ёзәҝзЁӢд№Ӣй—ҙзҡ„жЎҘжўҒпјҢиҝҷж ·е°ұеҸҜд»ҘдҪҝз”ЁеҶ…ж ёжҸҗдҫӣзҡ„зәҝзЁӢи°ғеәҰеҠҹиғҪеҸҠеӨ„зҗҶеҷЁжҳ е°„пјҢе№¶дё”з”ЁжҲ·зәҝзЁӢзҡ„зі»з»ҹи°ғз”ЁиҰҒйҖҡиҝҮиҪ»йҮҸзә§иҝӣзЁӢжқҘе®ҢжҲҗгҖӮиҝҷз§Қе°ұжҳҜеӨҡеҜ№еӨҡзҡ„зәҝзЁӢжЁЎеһӢгҖӮ

JavaзәҝзЁӢи°ғеәҰ

еҚҸеҗҢејҸи°ғеәҰпјҡзәҝзЁӢзҡ„жү§иЎҢдәӢд»¶з”ұзәҝзЁӢжң¬иә«жқҘжҺ§еҲ¶пјҢзәҝзЁӢжҠҠиҮӘе·ұзҡ„е·ҘдҪңжү§иЎҢе®ҢжҲҗпјҢдё»еҠЁйҖҡзҹҘзі»з»ҹеҲҮжҚўеҲ°еҸҰеӨ–дёҖдёӘзәҝзЁӢдёҠгҖӮзәҝзЁӢеҮәзҺ°йҳ»еЎһдјҡеҜјиҮҙиҝӣзЁӢйҳ»еЎһпјҢд»ҺиҖҢеҜјиҮҙзі»з»ҹеҙ©жәғгҖӮ

жҠўеҚ ејҸи°ғеәҰпјҡжҜҸдёӘзәҝзЁӢз”ұзі»з»ҹеҲҶй…Қжү§иЎҢж—¶й—ҙпјҢзәҝзЁӢзҡ„еҲҮжҚўдёҚз”ұзәҝзЁӢжң¬иә«еҶіе®ҡгҖӮзәҝзЁӢзҡ„жү§иЎҢж—¶й—ҙжҳҜеҸҜжҺ§зҡ„пјҢJavaжҸҗдҫӣдәҶ10з§ҚзәҝзЁӢдјҳе…Ҳзә§гҖӮ

зҠ¶жҖҒиҪ¬жҚў

JavaзәҝзЁӢжңү6з§ҚзҠ¶жҖҒпјҢеңЁд»»ж„Ҹзҡ„дёҖдёӘж—¶й—ҙзӮ№пјҢдёҖдёӘзәҝзЁӢеҸӘиғҪжңүдё”д»…жңүдёҖз§Қе…¶дёӯзҡ„зҠ¶жҖҒгҖӮ

  • ж–°е»ә(New)пјҡеҲӣе»әеҗҺжңӘеҗҜеҠЁзҡ„зәҝзЁӢзҠ¶жҖҒпјӣ
  • иҝҗиЎҢ(Runnable)пјҡеҢ…жӢ¬ж“ҚдҪңзі»з»ҹзәҝзЁӢзҠ¶жҖҒзҡ„Runningе’ҢReadyпјҢеӨ„дәҺжӯӨзҠ¶жҖҒзҡ„зәҝзЁӢеҸҜиғҪжӯЈеңЁиҝҗиЎҢпјҢд№ҹеҸҜиғҪжӯЈеңЁзӯүеҫ…ж“ҚдҪңзі»з»ҹеҲҶй…Қжү§иЎҢж—¶й—ҙпјӣ
  • ж— жңҹйҷҗзӯүеҫ…(Waiting)пјҡеӨ„дәҺиҝҷз§ҚзҠ¶жҖҒзҡ„зәҝзЁӢдёҚдјҡиў«еҲҶй…ҚеӨ„зҗҶеҷЁжү§иЎҢж—¶й—ҙпјҢе®ғд»¬иҰҒзӯүеҫ…е…¶е®ғзәҝзЁӢжҳҫзӨәе”ӨйҶ’гҖӮ
  • йҷҗжңҹзӯүеҫ…(Timed Waiting)пјҡеӨ„дәҺиҝҷз§ҚзҠ¶жҖҒзҡ„зәҝзЁӢдёҚдјҡиў«еҲҶй…ҚеӨ„зҗҶеҷЁж—¶й—ҙпјҢе®ғд»¬ж— йңҖзӯүеҫ…е…¶е®ғзәҝзЁӢжҳҫзӨәе”ӨйҶ’пјҢеңЁдёҖе®ҡж—¶й—ҙиҝҮеҗҺдјҡз”ұзі»з»ҹиҮӘеҠЁе”ӨйҶ’гҖӮ
  • йҳ»еЎһ(Blocked)пјҡзәҝзЁӢйҳ»еЎһпјҢеңЁзӯүеҫ…иҺ·еҸ–дёҖдёӘжҺ’д»–й”ҒгҖӮ
  • з»“жқҹ(Terminated)пјҡе·Із»ҲжӯўзәҝзЁӢзҡ„зәҝзЁӢзҠ¶жҖҒгҖӮ

JavaзәҝзЁӢе®үе…Ё

зәҝзЁӢд№ӢжүҖд»ҘдёҚе®үе…ЁпјҢжәҗдәҺеӨҡзәҝзЁӢеҜ№е…ұдә«еҸҳйҮҸи®ҝй—®пјҢJavaеӨҡзәҝзЁӢеҜ№е…ұдә«еҸҳйҮҸзҡ„ж“ҚдҪңеҸҜеҲҶдёәдёҚеҸҜеҸҳгҖҒз»қең°зәҝзЁӢе®үе…ЁгҖҒзӣёеҜ№зәҝзЁӢе®үе…ЁгҖҒзәҝзЁӢе…је®№е’ҢзәҝзЁӢеҜ№з«Ӣ(зәҝзЁӢе®үе…ЁзЁӢеәҰз”ұејәеҲ°ејұ)гҖӮ

  • зәҝзЁӢеҜ№з«Ӣпјҡжӯ»й”Ғ

дә’ж–ҘеҗҢжӯҘ

еҗҢжӯҘж—¶жҢҮеӨҡзәҝзЁӢе№¶еҸ‘и®ҝй—®е…ұдә«ж•°жҚ®ж—¶пјҢдҝқиҜҒе…ұдә«ж•°жҚ®еңЁеҗҢдёҖж—¶еҲ»еҸӘиғҪиў«дёҖжқЎ(жҲ–дёҖдәӣпјҢдҝЎеҸ·йҮҸ)зәҝзЁӢдҪҝз”ЁгҖӮдә’ж–Ҙж—¶е®һзҺ°еҗҢжӯҘзҡ„жүӢж®өпјҢдёҙз•ҢеҢәгҖҒдә’ж–ҘйҮҸе’ҢдҝЎеҸ·йҮҸйғҪжҳҜеёёи§Ғзҡ„дә’ж–Ҙе®һзҺ°ж–№ејҸгҖӮдә’ж–ҘжҳҜеӣ пјҢеҗҢжӯҘжҳҜжһңпјӣдә’ж–Ҙдёәж–№жі•пјҢеҗҢжӯҘдёәзӣ®зҡ„гҖӮ

Lockй”Ғе’Ңsynchronizedй”Ғ

synchronizedе’ҢReentrantLockй”Ғд№Ӣй—ҙзҡ„еҢәеҲ«

  1. жҖ§иғҪпјҡдҪҺе№¶еҸ‘жғ…еҶөдёӢпјҢsynchronizedдјҳдәҺReentrantLockпјҢеӣ дёәsynchronizedеҶ…зҪ®дәҶJVMзҡ„дјҳеҢ–жңәеҲ¶пјҢеҸҜд»Ҙе°ҶеҠ й”Ғе’Ңи§Јй”Ғж“ҚдҪңеҶ…иҒ”еҲ°д»Јз ҒдёӯгҖӮеңЁй«ҳе№¶еҸ‘жғ…еҶөдёӢпјҢReentrantLockжҖ§иғҪжӣҙеҘҪпјҢеӣ дёәе®ғж”ҜжҢҒе…¬е№ій”Ғе’Ңйқһе…¬е№ій”ҒдёӨз§ҚжЁЎејҸпјҢиҖҢдё”ж”ҜжҢҒдёӯж–ӯгҖҒи¶…ж—¶зӯүй«ҳзә§зү№жҖ§
  2. еҸҜдёӯж–ӯжҖ§пјҡReentrantLockж”ҜжҢҒдёӯж–ӯзӯүеҫ…й”Ғзҡ„зәҝзЁӢпјҢеҪ“дёҖдёӘзәҝзЁӢзӯүеҫ…й”Ғзҡ„иҝҮзЁӢдёӯпјҢеҸҜд»Ҙиў«е…¶д»–зәҝзЁӢдёӯж–ӯ()пјҢsynchronizedдёҚж”ҜжҢҒ
  3. е…¬е№іжҖ§пјҡеҪ“еӨҡдёӘзәҝзЁӢзӯүеҫ…еҗҢдёҖдёӘй”Ғж—¶пјҢReentrantLockеҸҜд»ҘдҝқиҜҒзӯүеҫ…ж—¶й—ҙжңҖй•ҝзҡ„зәҝзЁӢе…ҲиҺ·еҸ–еҲ°й”Ғ(е…¬е№ій”Ғ)пјҢиҖҢsynchronizedеҒҡдёҚеҲ°гҖӮй»ҳи®ӨжҳҜйқһе…¬е№ій”ҒпјҢдҪҝз”Ёе…¬е№ій”ҒпјҢжҖ§иғҪдјҡдёӢйҷҚ
  4. зӯүеҫ…йҖҡзҹҘжңәеҲ¶пјҡsynchronizedеҶ…зҪ®дәҶwaitгҖҒnotifyгҖҒnotifyAllзӯүж–№жі•пјҢе®һзҺ°зәҝзЁӢд№Ӣй—ҙзҡ„зӯүеҫ…е”ӨйҶ’жңәеҲ¶пјҢReentrantLockйңҖиҰҒеҖҹеҠ©ConditionеҜ№иұЎжқҘе®һзҺ°(lockй”Ғй”ҒдҪҸзҡ„дәҶзҡ„еҜ№иұЎпјҢжІЎжңүи§Јй”ҒпјҢе…¶д»–зәҝзЁӢж— жі•и®ҝй—®пјҢеҸҜд»ҘдҪҝз”Ёlock.newCondition().await()йҮҠж”ҫй”ҒпјҢи®©е…¶д»–зәҝзЁӢи®ҝй—®пјҢдҪҶжҳҜзәҝзЁӢзҡ„е”ӨйҶ’еҝ…йЎ»иҰҒиҝҷдёӘзәҝзЁӢдҪҝз”ЁдәҶlockй”Ғ(еҰӮжһңйҮҠж”ҫй”Ғд№ӢеҗҺжүҚе”ӨйҶ’зәҝзЁӢпјҢдјҡеҮәзҺ°IllegalMonitorStateExceptionејӮеёё)пјҢиҰҒеңЁlock.unlock()д№ӢеүҚи°ғз”Ёlock.newCondition().signal()жқҘе”ӨйҶ’пјҢеҗҰеҲҷеҮәзҺ°ејӮеёё)

дёӯж–ӯжҖ§

йқһйҳ»еЎһеҗҢжӯҘ

дә’ж–ҘеҗҢжӯҘйқўдёҙзҡ„дё»иҰҒй—®йўҳжҳҜиҝӣиЎҢзәҝзЁӢйҳ»еЎһе’Ңе”ӨйҶ’жүҖеёҰжқҘзҡ„жҖ§иғҪејҖй”ҖпјҢеӣ жӯӨд№ҹз§°дёәйҳ»еЎһеҗҢжӯҘгҖӮ

Javaеј•е…ҘдәҶж–°зҡ„ж–№ејҸпјҢCAS(жҜ”иҫғе№¶дәӨжҚў)

CASйңҖиҰҒдёүдёӘеҸӮж•°пјҡеҶ…еӯҳдҪҚзҪ®гҖҒж—§зҡ„йў„жңҹеҖјгҖҒеҮҶеӨҮи®ҫзҪ®зҡ„ж–°еҖјгҖӮ

иҝҷйҮҢvar1е’Ңvar2д»ЈиЎЁеҶ…еӯҳдҪҚзҪ®пјӣ

var5д»ЈиЎЁж—§зҡ„йў„жңҹеҖјпјӣ

var5+var4д»ЈиЎЁеҮҶеӨҮи®ҫзҪ®зҡ„ж–°еҖјгҖӮ

ж— еҗҢжӯҘж–№жЎҲ

ж–№жі•дёӯжң¬иә«е°ұжІЎжңүи®ҫзҪ®е…ұдә«ж•°жҚ®гҖӮ

зәҝзЁӢжң¬ең°еӯҳеӮЁпјҢе…ұдә«еҸҳйҮҸжҳҜеҗҰиғҪдҝқиҜҒеҸӘеңЁеҗҢдёҖдёӘзәҝзЁӢдёӯжү§иЎҢгҖӮ

ж¶ҲжҒҜйҳҹеҲ—дјҡе°Ҷж¶Ҳиҙ№иҝҮзЁӢжҺ§еҲ¶еңЁдёҖдёӘзәҝзЁӢдёӯе®ҢжҲҗгҖӮ

й”ҒдјҳеҢ–

иҮӘж—Ӣй”Ғе’ҢиҮӘйҖӮеә”иҮӘж—Ӣй”Ғ

иҮӘж—Ӣй”ҒеңЁзӯүеҫ…зҡ„иҝҮзЁӢдёӯдјҡиҖ—иҙ№еӨ„зҗҶеҷЁзҡ„жү§иЎҢж—¶й—ҙгҖӮеҸҜд»ҘйҖҡиҝҮи®ҫзҪ®иҮӘж—Ӣзҡ„ж¬Ўж•°жқҘжҺ§еҲ¶йҖҖеҮәиҮӘж—Ӣзҡ„зҠ¶жҖҒ(жҢӮиө·зәҝзЁӢ)гҖӮ

иҮӘйҖӮеә”иҮӘж—Ӣй”Ғзҡ„зӣ®зҡ„жҳҜпјҡиҮӘйҖӮеә”иҮӘж—Ӣй”Ғзҡ„зӯүеҫ…ж—¶й—ҙдёҚеҶҚжҳҜеӣәе®ҡзҡ„дәҶпјҢиҖҢжҳҜз”ұдёҠдёҖж¬ЎеңЁеҗҢдёҖдёӘй”ҒдёҠзҡ„иҮӘж—Ӣж—¶й—ҙеҸҠй”Ғзҡ„жӢҘжңүиҖ…зҡ„зҠ¶жҖҒжқҘеҶіе®ҡгҖӮеҰӮжһңеңЁеҗҢдёҖдёӘй”ҒеҜ№иұЎдёҠпјҢиҮӘж—Ӣзӯүеҫ…еҲҡеҲҡжҲҗеҠҹиҺ·еҫ—иҝҮй”ҒпјҢе№¶дё”жҢҒжңүй”Ғзҡ„зәҝзЁӢжӯЈеңЁиҝҗиЎҢдёӯпјҢйӮЈд№ҲиҷҡжӢҹжңәе°ұдјҡи®ӨдёәжӯӨж¬ЎиҮӘж—Ӣд№ҹеҫҲжңүеҸҜиғҪеҶҚж¬ЎжҲҗеҠҹпјҢиҝӣиҖҢиҝҗиЎҢиҮӘж—Ӣзӯүеҫ…жҢҒз»ӯж—¶й—ҙзӣёеҜ№иҫғй•ҝзҡ„ж—¶й—ҙпјҢеҰӮжҢҒз»ӯ100ж¬Ўзҡ„иҮӘж—ӢгҖӮеҰӮпјҢеҜ№дәҺжҹҗдёӘй”ҒпјҢиҮӘж—ӢеҫҲе°‘жҲҗеҠҹиҺ·еҸ–й”ҒпјҢйӮЈд№Ҳд»ҘеҗҺиҺ·еҸ–иҝҷдёӘй”Ғж—¶е°ҶеҫҲжңүеҸҜиғҪзңҒеҺ»иҮӘж—ӢиҝҮзЁӢпјҢд»ҘйҒҝе…ҚжөӘиҙ№еӨ„зҗҶеҷЁж—¶й—ҙгҖӮ

й”Ғж¶ҲйҷӨ

дҫӢеҰӮJavaдёӯStringеӯ—з¬ҰдёІзҡ„жӢјжҺҘпјҢеҸҜиғҪдјҡиҪ¬жҚўдёәStringBufferзҡ„append()жЁЎејҸпјҢиҝҷдёӘжҳҜдёҖдёӘеҗҢжӯҘж–№жі•пјҢиҝҷйҮҢзҡ„еҗҢжӯҘд»Јз Ғж №жң¬дёҚеӯҳеңЁе…ұдә«ж•°жҚ®з«һдәүзҡ„й—®йўҳпјҢжүҖд»ҘиҝӣиЎҢдәҶй”Ғж¶ҲйҷӨзҡ„дјҳеҢ–зӯ–з•Ҙ

й”ҒзІ—еҢ–

еҜ№дәҺеҗҢдёҖдёӘеҜ№иұЎдјҡиҝӣиЎҢеӨҡж¬Ўзҡ„еҠ й”Ғе’Ңи§Јй”ҒпјҢиҷҡжӢҹжңәиғҪжҺ§еҲ¶еҸӘеҠ й”ҒдёҖж¬ЎпјҢдҫӢеҰӮStringBufferзҡ„append()ж–№жі•еӨҡж¬Ўжү§иЎҢгҖӮ

еҒҸеҗ‘й”Ғ

  • еҒҸеҗ‘й”ҒжҳҜдёәдәҶи§ЈеҶіеңЁж— з«һдәүзҡ„жғ…еҶөдёӢеҜ№й”Ғзҡ„жҖ§иғҪжҚҹиҖ—й—®йўҳиҖҢеј•е…Ҙзҡ„жңәеҲ¶пјӣ
  • еҪ“дёҖдёӘзәҝзЁӢиҺ·еҸ–й”Ғе№¶дё”жІЎжңүе…¶д»–зәҝзЁӢз«һдәүж—¶пјҢеҒҸеҗ‘й”Ғдјҡе°Ҷй”Ғж Үи®°дёәеҒҸеҗ‘дәҺиҜҘзәҝзЁӢпјҢе№¶дё”д№ӢеҗҺиҜҘзәҝзЁӢеҶҚж¬ЎиҺ·еҸ–й”Ғж—¶еҸҜд»ҘзӣҙжҺҘиҺ·еҫ—пјҢиҖҢдёҚйңҖиҰҒиҝӣиЎҢеҗҢжӯҘж“ҚдҪңпјӣ
  • еҰӮжһңеңЁеӨҡзәҝзЁӢзҺҜеўғдёӢпјҢжІЎжңүе…¶д»–зәҝзЁӢдәүеӨәй”Ғж—¶пјҢдҪҝз”ЁеҒҸеҗ‘й”ҒеҸҜд»ҘеҮҸе°‘дёҚеҝ…иҰҒзҡ„еҗҢжӯҘж“ҚдҪңпјҢжҸҗй«ҳзЁӢеәҸжҖ§иғҪ

еңЁж— з«һдәүзҡ„жғ…еҶөдёӢпјҢжҠҠж•ҙдёӘеҗҢжӯҘйғҪж¶ҲйҷӨжҺүпјҢиҝһCASж“ҚдҪңйғҪдёҚйңҖиҰҒгҖӮ

иҪ»йҮҸзә§й”Ғ

  • иҪ»йҮҸзә§й”ҒжҳҜдёәдәҶи§ЈеҶіеңЁз«һдәүдёҚжҝҖзғҲзҡ„жғ…еҶөдёӢеҜ№й”ҒжҖ§иғҪжҚҹиҖ—й—®йўҳиҖҢеј•е…Ҙзҡ„жңәеҲ¶пјӣ
