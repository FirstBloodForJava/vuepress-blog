RabbitMQ

MQ(Message Queue)ж¶ҲжҒҜйҳҹеҲ—пјҢе…ёеһӢзҡ„з”ҹдә§иҖ…ж¶Ҳиҙ№иҖ…жЁЎејҸпјҢз”ҹдә§иҖ…дёҚж–ӯеҗ‘ж¶ҲжҒҜйҳҹеҲ—з”ҹдә§ж¶ҲжҒҜпјҢж¶Ҳиҙ№иҖ…дёҚж–ӯд»Һж¶ҲжҒҜйҳҹеҲ—дёӯиҺ·еҸ–ж¶ҲжҒҜгҖӮеӣ дёәж¶ҲжҒҜзҡ„з”ҹдә§е’Ңж¶Ҳиҙ№йғҪжҳҜејӮжӯҘзҡ„пјҢиҖҢдё”еҸӘе…іеҝғж¶ҲжҒҜзҡ„еҸ‘йҖҒе’ҢжҺҘж”¶пјҢжІЎжңүдёҡеҠЎйҖ»иҫ‘зҡ„дҫөе…ҘпјҢиҪ»жқҫе®һзҺ°дәҶзі»з»ҹд№Ӣй—ҙзҡ„и§ЈиҖҰеҗҲгҖӮйҖҡиҝҮй«ҳж•Ҳзҡ„еҸҜйқ зҡ„ж¶ҲжҒҜдј йҖ’жңәеҲ¶иҝӣиЎҢзі»з»ҹд№Ӣй—ҙзҡ„йҖҡдҝЎжқҘе®һзҺ°еҲҶеёғејҸзі»з»ҹгҖӮ

RabbitMQжҳҜе®һзҺ°дәҶй«ҳзә§ж¶ҲжҒҜйҳҹеҲ—еҚҸи®®пјҲAMQPпјүзҡ„ејҖжәҗж¶ҲжҒҜд»ЈзҗҶиҪҜд»¶пјҲдәҰз§°йқўеҗ‘ж¶ҲжҒҜзҡ„дёӯй—ҙд»¶пјүгҖӮRabbitMQжңҚеҠЎеҷЁжҳҜз”ЁErlangиҜӯиЁҖзј–еҶҷзҡ„пјҢиҖҢйӣҶзҫӨе’Ңж•…йҡңиҪ¬з§»жҳҜжһ„е»әеңЁејҖж”ҫз”өдҝЎе№іеҸ°жЎҶжһ¶дёҠзҡ„гҖӮжүҖжңүдё»иҰҒзҡ„зј–зЁӢиҜӯиЁҖеқҮжңүдёҺд»ЈзҗҶжҺҘеҸЈйҖҡи®Ҝзҡ„е®ўжҲ·з«Ҝеә“гҖӮ

е®ҳзҪ‘пјҡhttps://www.rabbitmq.com/

javaе®ўжҲ·з«Ҝж–ҮжЎЈпјҡhttps://www.rabbitmq.com/api-guide.html

  • и§ЈиҖҰпјҡдј з»ҹзҡ„ејҖеҸ‘жЁЎејҸдёӯпјҢеҗ„дёӘжЁЎеқ—д№Ӣй—ҙзӣёдә’и°ғз”ЁпјҢж•°жҚ®е…ұдә«пјҢжҜҸдёӘжЁЎеқ—йғҪйңҖиҰҒе…іжіЁе…¶д»–жЁЎеқ—жҳҜеҗҰжҢӮжҺүгҖӮдҪҝз”Ёж¶ҲжҒҜйҳҹеҲ—пјҢеҸҜд»Ҙе°Ҷе…ұдә«зҡ„ж•°жҚ®ж”ҫеңЁж¶ҲжҒҜйҳҹеҲ—дёӯпјҢж–°еўһдёҡеҠЎжЁЎеқ—пјҢдёҡеҠЎжЁЎеқ—еҸҜд»Ҙи®ўйҳ…иҜҘж¶ҲжҒҜпјҢеҜ№еҺҹжңүзҡ„зі»з»ҹдёҡеҠЎжІЎжңүд»»дҪ•еҪұе“ҚпјҢйҷҚдҪҺдәҶеҗ„дёӘзі»з»ҹд№Ӣй—ҙзҡ„иҖҰеҗҲеәҰпјҢжҸҗй«ҳзі»з»ҹзҡ„еҸҜжү©еұ•жҖ§гҖӮ
  • жөҒйҮҸеүҠй”Ӣпјҡе°ҶеҜ№жңҚеҠЎеҷЁзҡ„иҜ·жұӮйҖҡиҝҮж¶ҲжҒҜйҳҹеҲ—еҶҚж¬ЎжӢҰжҲӘеӨ„зҗҶ

  • * пјҲstarпјү can substitute for exactly one word
  • # пјҲhashпјүcan substitute for zero or more words.
Topic exchange is powerful and can behave like other exchanges.

When a queue is bound with "#" (hash) binding key - it will receive all the messages, regardless of the routing key - like in fanout exchange.

When special characters, "*" (star) and "#" (hash), aren't used in bindings, the topic exchange will behave just like a direct one.

	}

		.to(template::setReplyTimeout);

MVCC

MVCCпјҲMulti-Version Concurrency ControlеӨҡзүҲжң¬е№¶еҸ‘жҺ§еҲ¶пјүжҳҜдёҖз§Қз”ЁдәҺж•°жҚ®еә“з®ЎзҗҶзі»з»ҹпјҲDBMSпјүдёӯе®һзҺ°е№¶еҸ‘жҺ§еҲ¶зҡ„жҠҖжңҜгҖӮMVCC жңәеҲ¶йҖҡиҝҮжҜҸдёӘдәӢеҠЎеҲҶй…Қе”ҜдёҖзҡ„дәӢеҠЎж ҮиҜҶз¬ҰпјҲTransaction IDпјҢTIDпјүжқҘе…Ғи®ёеӨҡдёӘдәӢеҠЎе№¶еҸ‘ең°и®ҝй—®ж•°жҚ®еә“пјҢд»ҺиҖҢйҒҝе…ҚдәҶй”Ғе®ҡиө„жәҗд»ҘйҳІжӯўеҶІзӘҒзҡ„дј з»ҹе№¶еҸ‘жҺ§еҲ¶ж–№жі•зҡ„дёҖдәӣзјәзӮ№гҖӮ

еңЁ MVCC дёӯпјҢжҜҸдёӘдәӢеҠЎеңЁејҖе§Ӣж—¶йғҪдјҡеҲӣе»әдёҖдёӘж—¶й—ҙжҲіпјҢе№¶еңЁе…¶жү§иЎҢжңҹй—ҙдҝқжҢҒдёҚеҸҳгҖӮжҜҸдёӘж•°жҚ®еә“еҜ№иұЎпјҲеҰӮиЎЁгҖҒиЎҢжҲ–йЎөпјүйғҪдјҡеҲҶй…ҚдёҖдёӘжҲ–еӨҡдёӘзүҲжң¬еҸ·гҖӮеҪ“дәӢеҠЎи®ҝй—®ж•°жҚ®еә“еҜ№иұЎж—¶пјҢе®ғе°ҶеҸӘиғҪи®ҝй—®ж—©дәҺиҜҘж—¶й—ҙжҲізҡ„зүҲжң¬гҖӮ

MVCC жңәеҲ¶дёӯзҡ„иҜ»ж“ҚдҪңдёҺеҶҷж“ҚдҪңд№Ӣй—ҙеӯҳеңЁдёҚеҗҢзҡ„и§„еҲҷгҖӮеңЁиҜ»ж“ҚдҪңдёӯпјҢеҸӘжңүж—©дәҺиҜҘж—¶й—ҙжҲізҡ„е·ІжҸҗдәӨзүҲжң¬жүҚдјҡиў«иҜ»еҸ–пјҢеӣ жӯӨиҜ»ж“ҚдҪңдёҚдјҡеҸ—еҲ°жӯЈеңЁжү§иЎҢзҡ„е№¶еҸ‘еҶҷж“ҚдҪңзҡ„е№Іжү°гҖӮеңЁеҶҷж“ҚдҪңдёӯпјҢдәӢеҠЎдјҡеҲӣе»әж–°зүҲжң¬пјҢе№¶е°Ҷе…¶еҶҷе…ҘзЈҒзӣҳгҖӮеҰӮжһңеҸҰдёҖдёӘдәӢеҠЎжӯЈеңЁи®ҝй—®ж—§зүҲжң¬пјҢеҲҷиҜҘдәӢеҠЎе°Ҷз»§з»ӯиҜ»еҸ–ж—§зүҲжң¬пјҢиҖҢдёҚжҳҜиў«ж–°зүҲжң¬жүҖиҰҶзӣ–гҖӮ

MVCC жңәеҲ¶зҡ„дјҳзӮ№жҳҜжҸҗй«ҳдәҶж•°жҚ®еә“зі»з»ҹзҡ„е№¶еҸ‘жҖ§иғҪпјҢеӣ дёәеӨҡдёӘдәӢеҠЎеҸҜд»ҘеҗҢж—¶иҜ»еҸ–е’Ңдҝ®ж”№ж•°жҚ®еә“еҜ№иұЎпјҢиҖҢдёҚдјҡдә’зӣёе№Іжү°гҖӮеҗҢж—¶пјҢMVCC жңәеҲ¶д№ҹйҷҚдҪҺдәҶжӯ»й”Ғзҡ„йЈҺйҷ©пјҢеӣ дёәдәӢеҠЎдёҚйңҖиҰҒзӯүеҫ…е…¶д»–дәӢеҠЎйҮҠж”ҫй”Ғе®ҡзҡ„иө„жәҗжүҚиғҪз»§з»ӯжү§иЎҢгҖӮ

дҪҶжҳҜпјҢMVCC жңәеҲ¶д№ҹжңүдёҖдәӣзјәзӮ№гҖӮдҫӢеҰӮпјҢе®ғдјҡеўһеҠ ж•°жҚ®еә“зі»з»ҹзҡ„еӯҳеӮЁжҲҗжң¬пјҢеӣ дёәжҜҸдёӘзүҲжң¬йғҪйңҖиҰҒйўқеӨ–зҡ„еӯҳеӮЁз©әй—ҙгҖӮжӯӨеӨ–пјҢеңЁжҹҗдәӣжғ…еҶөдёӢпјҢMVCC жңәеҲ¶еҸҜиғҪдјҡеҜјиҮҙдёҖдәӣеҘҮжҖӘзҡ„зҺ°иұЎпјҢдҫӢеҰӮдёҚеҸҜйҮҚеӨҚиҜ»жҲ–е№»иҜ»пјҢиҝҷдәӣйғҪжҳҜеӣ дёәеңЁдёҚеҗҢзҡ„ж—¶й—ҙжҲідёӢиҜ»еҸ–дәҶдёҚеҗҢзҡ„зүҲжң¬жүҖеҜјиҮҙзҡ„гҖӮ

MVCCпјҲMulti-Version Concurrency ControlпјүжҳҜдёҖз§Қж•°жҚ®еә“е№¶еҸ‘жҺ§еҲ¶жҠҖжңҜпјҢе®ғе…Ғи®ёеӨҡдёӘдәӢеҠЎеҗҢж—¶иҜ»еҸ–ж•°жҚ®еә“дёӯзҡ„еҗҢдёҖж•°жҚ®пјҢиҖҢдёҚдјҡеҸ‘з”ҹж•°жҚ®еҶІзӘҒе’ҢдёўеӨұжӣҙж–°зҡ„й—®йўҳгҖӮMVCCжңҖж—©жҳҜеңЁPostgreSQLж•°жҚ®еә“дёӯе®һзҺ°зҡ„пјҢзҺ°еңЁе·Із»Ҹиў«е№ҝжіӣеә”з”ЁдәҺи®ёеӨҡж•°жҚ®еә“зі»з»ҹдёӯгҖӮ

MVCCзҡ„еҹәжң¬жҖқжғіжҳҜпјҢдёәжҜҸдёӘдәӢеҠЎеҲҶй…ҚдёҖдёӘе”ҜдёҖзҡ„ж—¶й—ҙжҲіпјҲжҲ–зүҲжң¬еҸ·пјүпјҢе№¶е°ҶжҜҸдёӘж•°жҚ®иЎҢзҡ„жүҖжңүзүҲжң¬йғҪдҝқеӯҳеңЁж•°жҚ®еә“дёӯгҖӮжҜҸдёӘдәӢеҠЎеҸӘиғҪзңӢеҲ°ж—©дәҺиҮӘе·ұж—¶й—ҙжҲіпјҲжҲ–зүҲжң¬еҸ·пјүзҡ„ж•°жҚ®иЎҢзүҲжң¬гҖӮеҪ“дёҖдёӘдәӢеҠЎдҝ®ж”№дёҖиЎҢж•°жҚ®ж—¶пјҢе®ғдјҡеңЁиҜҘж•°жҚ®иЎҢзҡ„зүҲжң¬й“ҫдёӯеҲӣе»әдёҖдёӘж–°зүҲжң¬пјҢе№¶е°ҶиҮӘе·ұзҡ„ж—¶й—ҙжҲіпјҲжҲ–зүҲжң¬еҸ·пјүеҲҶй…Қз»ҷиҝҷдёӘж–°зүҲжң¬гҖӮеӣ жӯӨпјҢдёҚеҗҢдәӢеҠЎзҡ„дҝ®ж”№дјҡеҪўжҲҗдёҚеҗҢзҡ„зүҲжң¬й“ҫгҖӮ

еҪ“дёҖдёӘдәӢеҠЎиҰҒиҜ»еҸ–дёҖдёӘж•°жҚ®иЎҢж—¶пјҢе®ғдјҡжҹҘжүҫиҜҘж•°жҚ®иЎҢзҡ„зүҲжң¬й“ҫпјҢжүҫеҲ°жңҖж–°зҡ„ж—©дәҺиҮӘе·ұж—¶й—ҙжҲіпјҲжҲ–зүҲжң¬еҸ·пјүзҡ„зүҲжң¬пјҢе№¶иҜ»еҸ–е…¶дёӯзҡ„ж•°жҚ®гҖӮеҰӮжһңж•°жҚ®иЎҢзҡ„жңҖж–°зүҲжң¬ж—©дәҺдәӢеҠЎзҡ„ж—¶й—ҙжҲіпјҲжҲ–зүҲжң¬еҸ·пјүпјҢеҲҷж„Ҹе‘ізқҖе…¶д»–дәӢеҠЎе·Із»Ҹдҝ®ж”№дәҶиҜҘж•°жҚ®иЎҢпјҢиҜҘдәӢеҠЎйңҖиҰҒеӣһж»ҡе№¶йҮҚж–°иҜ»еҸ–ж•°жҚ®гҖӮ
