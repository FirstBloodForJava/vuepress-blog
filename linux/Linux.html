д»Җд№ҲжҳҜLinux

linuxжҳҜдёҖдёӘејҖжәҗгҖҒе…Қиҙ№зҡ„ж“ҚдҪңзі»з»ҹпјҢе…¶зЁіе®ҡжҖ§гҖҒе®үе…ЁжҖ§гҖҒеӨ„зҗҶеӨҡе№¶еҸ‘иғҪеҠӣе·Із»Ҹеҫ—еҲ°дёҡз•Ңзҡ„и®ӨеҸҜпјҢзӣ®еүҚеӨ§еӨҡж•°дјҒдёҡзә§еә”з”Ёз”ҡиҮіжҳҜйӣҶзҫӨйЎ№зӣ®йғҪйғЁзҪІиҝҗиЎҢеңЁlinuxж“ҚдҪңзі»з»ҹд№ӢдёҠпјҢеҫҲеӨҡиҪҜд»¶е…¬еҸёиҖғиҷ‘еҲ°ејҖеҸ‘жҲҗжң¬йғҪйҰ–йҖүlinuxпјҢеңЁдёӯеӣҪиҪҜд»¶е…¬еҸёеҫ—еҲ°е№ҝжіӣзҡ„дҪҝз”ЁгҖӮ

LinuxеҶ…ж ёkernelжңҖеҲқжҳҜз”ұиҠ¬е…°дәәжқҺзәіж–ҜВ·жүҳз“Ұе…№пјҲLinus TorvaldsпјүеңЁиө«е°”иҫӣеҹәеӨ§еӯҰдёҠеӯҰж—¶еҮәдәҺдёӘдәәзҲұеҘҪиҖҢзј–еҶҷзҡ„гҖӮеңЁ1991е№ҙ10жңҲ5ж—Ҙз¬¬дёҖж¬ЎжӯЈејҸеҗ‘еӨ–е…¬еёғгҖӮ

еңЁlinuxеҸ‘еёғд№ӢеүҚпјҢжңүдёҖз§Қж“ҚдҪңзі»з»ҹеҸ«unixпјҢз”ұдәҺд»·ж јжҳӮиҙөдёҚејҖж”ҫжәҗз ҒпјҢжүҖд»ҘжқҺзәіж–ҜВ·жүҳз“Ұе…№пјҲLinus TorvaldsпјүеҶіеҝғиҰҒејҖеҸ‘иҮӘе·ұе…Қиҙ№зҡ„ж“ҚдҪңзі»з»ҹпјҢLinuxеҖҹйүҙдәҶunixзҡ„жҖқжғіпјҢдҪҶжІЎжңүдёҖиЎҢunixзҡ„д»Јз ҒпјҢlinuxзі»з»ҹжҳҜд»Һunixзі»з»ҹеҸ‘еұ•еҮәжқҘзҡ„гҖӮ Linux иӢұж–Үи§ЈйҮҠдёә Linux is not UnixгҖӮзҺ°еңЁдёҡз•ҢжңүдёҖз§ҚиҜҙжі•еҸ«пјҡзұ»unix

ж“ҚдҪңзі»з»ҹпјҲOperating SystemпјҢз®Җз§°OSпјүжҳҜз®ЎзҗҶе’ҢжҺ§еҲ¶и®Ўз®—жңәзЎ¬д»¶дёҺиҪҜд»¶иө„жәҗзҡ„и®Ўз®—жңәзЁӢеәҸпјҢжҳҜзӣҙжҺҘиҝҗиЎҢеңЁвҖңиЈёжңәвҖқдёҠзҡ„жңҖеҹәжң¬зҡ„зі»з»ҹиҪҜд»¶пјҢд»»дҪ•е…¶д»–иҪҜд»¶йғҪеҝ…йЎ»еңЁж“ҚдҪңзі»з»ҹзҡ„ж”ҜжҢҒдёӢжүҚиғҪиҝҗиЎҢгҖӮ

ж“ҚдҪңзі»з»ҹжҳҜз”ЁжҲ·е’Ңи®Ўз®—жңәзҡ„жҺҘеҸЈпјҢеҗҢж—¶д№ҹжҳҜи®Ўз®—жңәзЎ¬д»¶е’Ңе…¶д»–иҪҜд»¶зҡ„жҺҘеҸЈгҖӮж“ҚдҪңзі»з»ҹзҡ„еҠҹиғҪеҢ…жӢ¬з®ЎзҗҶи®Ўз®—жңәзі»з»ҹзҡ„зЎ¬д»¶гҖҒиҪҜд»¶еҸҠж•°жҚ®иө„жәҗпјҢжҺ§еҲ¶зЁӢеәҸиҝҗиЎҢпјҢж”№е–„дәәжңәз•ҢйқўпјҢдёәе…¶е®ғеә”з”ЁиҪҜд»¶жҸҗдҫӣж”ҜжҢҒзӯүгҖӮе®һйҷ…дёҠпјҢз”ЁжҲ·жҳҜдёҚз”ЁжҺҘи§Ұж“ҚдҪңзі»з»ҹзҡ„пјҢж“ҚдҪңзі»з»ҹз®ЎзҗҶзқҖи®Ўз®—жңәзЎ¬д»¶иө„жәҗпјҢеҗҢж—¶жҢүзқҖеә”з”ЁзЁӢеәҸзҡ„иө„жәҗиҜ·жұӮпјҢдёәе…¶еҲҶй…Қиө„жәҗпјҢеҰӮпјҡеҲ’еҲҶCPUж—¶й—ҙпјҢеҶ…еӯҳз©әй—ҙзҡ„ејҖиҫҹпјҢи°ғз”Ёжү“еҚ°жңәзӯүгҖӮ

ctstate RELATED,ESTABLISHEDи§„еҲҷжҳҜз”ЁдәҺ iptables йҳІзҒ«еўҷи§„еҲҷдёӯзҡ„иҝһжҺҘи·ҹиёӘзҠ¶жҖҒпјҲconnection tracking stateпјүзҡ„и®ҫзҪ®гҖӮиҝҷдёӘи§„еҲҷз”ЁдәҺе…Ғи®ёдёҺзҺ°жңүиҝһжҺҘжңүе…ізҡ„е’Ңе·Із»Ҹе»әз«Ӣзҡ„ж•°жҚ®еҢ…йҖҡиҝҮйҳІзҒ«еўҷгҖӮиҝҷжҳҜдёҖдёӘеёёи§Ғзҡ„и®ҫзҪ®пјҢзЎ®дҝқе·Із»Ҹе»әз«Ӣзҡ„иҝһжҺҘд»ҘеҸҠдёҺиҝҷдәӣиҝһжҺҘзӣёе…ізҡ„ж•°жҚ®еҢ…иғҪеӨҹиҮӘз”ұйҖҡиҝҮйҳІзҒ«еўҷпјҢиҖҢдёҚеҸ—йўқеӨ–зҡ„йҷҗеҲ¶гҖӮиҝҷеҜ№дәҺз»ҙжҠӨиҝһжҺҘзҠ¶жҖҒе’ҢзЎ®дҝқзӣёе…ізҡ„ж•°жҚ®еҢ…иғҪеӨҹжӯЈеёёдј иҫ“йқһеёёйҮҚиҰҒпјҢжҜ”еҰӮеңЁзҪ‘з»ңдјҡиҜқдёӯзҡ„иҜ·жұӮе’Ңе“Қеә”д№Ӣй—ҙгҖӮ

  • ctstateпјҡиҝһжҺҘи·ҹиёӘжЁЎеқ—зҡ„е…ій”®еӯ—пјҢз”ЁдәҺеҢ№й…Қзү№е®ҡзҡ„иҝһжҺҘзҠ¶жҖҒ
  • RELATEDпјҡиЎЁзӨәеҢ№й…ҚдёҺзҺ°жңүиҝһжҺҘжңүе…ізҡ„ж•°жҚ®еҢ…гҖӮдҫӢеҰӮпјҢеҰӮжһңдҪ зҡ„зі»з»ҹдёҠжңүдёҖдёӘ FTP иҝһжҺҘпјҢйӮЈд№ҲдёҺ FTP жҺ§еҲ¶иҝһжҺҘзӣёе…ізҡ„ж•°жҚ®еҢ…д№ҹдјҡиў«еҢ№й…ҚгҖӮиҝҷзЎ®дҝқзӣёе…ізҡ„ж•°жҚ®еҢ…иғҪеӨҹйҖҡиҝҮйҳІзҒ«еўҷгҖӮ
  • ESTABLISHEDпјҡиЎЁзӨәеҢ№й…Қе·Із»Ҹе»әз«Ӣзҡ„иҝһжҺҘзҡ„ж•°жҚ®еҢ…гҖӮдёҖж—ҰиҝһжҺҘз»ҸиҝҮиҝһжҺҘе»әз«Ӣзҡ„жҸЎжүӢйҳ¶ж®өпјҢеҗҺз»ӯзҡ„ж•°жҚ®еҢ…йғҪиў«и®ӨдёәжҳҜе·Іе»әз«Ӣзҡ„иҝһжҺҘгҖӮ

reject-with icmp-port-unreachableи§„еҲҷжҳҜ iptables йҳІзҒ«еўҷи§„еҲҷдёӯз”ЁдәҺжӢ’з»қиҝһжҺҘзҡ„и®ҫзҪ®пјҢеҪ“иҝһжҺҘиў«жӢ’з»қж—¶пјҢзі»з»ҹе°Ҷиҝ”еӣһ ICMP й”ҷиҜҜж¶ҲжҒҜпјҢе…·дҪ“ең°иҜҙжҳҜ ICMP Port Unreachable й”ҷиҜҜж¶ҲжҒҜгҖӮ

иҝҷдёӘи§„еҲҷз”ЁдәҺжӢ’з»қиҝһжҺҘпјҢе№¶еҗ‘еҸ‘йҖҒж–№иҝ”еӣһ ICMP Port Unreachable й”ҷиҜҜж¶ҲжҒҜгҖӮеҪ“жңүиҝһжҺҘе°қиҜ•еҲ°иҫҫиў«йҳІзҒ«еўҷжӢ’з»қзҡ„з«ҜеҸЈж—¶пјҢзі»з»ҹе°ҶеҸ‘йҖҒиҝҷдёӘй”ҷиҜҜж¶ҲжҒҜпјҢйҖҡзҹҘеҸ‘йҖҒж–№зӣ®ж Үз«ҜеҸЈдёҚеҸҜиҫҫгҖӮ

  • reject-withпјҡжӢ’з»қиҝһжҺҘж—¶йҮҮеҸ–зҡ„еҠЁдҪңпјҢе№¶жҢҮе®ҡжӢ’з»қиҝһжҺҘж—¶иҝ”еӣһзҡ„ ICMP й”ҷиҜҜж¶ҲжҒҜзұ»еһӢпјӣ
  • icmp-port-unreachableпјҡиЎЁзӨәеҪ“иҝһжҺҘиў«жӢ’з»қж—¶иҝ”еӣһ ICMP Port Unreachable й”ҷиҜҜж¶ҲжҒҜгҖӮиҝҷдёӘй”ҷиҜҜж¶ҲжҒҜйҖҡеёёиЎЁзӨәзӣ®ж Үз«ҜеҸЈдёҚеҸҜиҫҫгҖӮ
